COVID TRACKER

***********SOFTWARE USED**************
1. python (https://www.python.org/downloads/)
2. node js  (https://nodejs.org/en/download/)
3. react js
4. sublime text (https://www.sublimetext.com/3)
5. phpmyadmin (https://www.phpmyadmin.net/downloads/)
6. mysql (https://dev.mysql.com/downloads/installer/)

***********PACKAGES INSTALLATION**********

Go to command prompt and write "pip install flask" and "pip install requests"
to run the flask project

***********TO RUN PROJECT*****************

1) to start the flask server go to root directory of the project
open cmd at that location and write "python app.py"

2) to run the react implementation go to root directory of the react project
and write "npm install" to install the dependencies then write "npm run start"

3) to run the php implementation save the project folder in apache24->htdocs
then go to browser and type localhost you will get the list of folders in htdocs
select the project folder
